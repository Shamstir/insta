# 📸 Insta

A simple Instagram clone built with **Flutter** and **Firebase** for learning and personal projects.

## 🚀 Features
- Photo sharing
- Likes & comments
- User authentication
- User profiles
- Realtime feed

## 🛠️ Tech Stack
- Flutter
- Firebase Auth
- Firebase Firestore
- Firebase Storage

## 🔧 How to Run
```bash
git clone https://github.com/your-username/insta.git
cd insta
flutter pub get
flutter run
